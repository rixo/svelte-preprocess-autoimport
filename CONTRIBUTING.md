# Contributing

## Install

```bash
git clone git@github.com:rixo/svelte-preprocess-autoimport.git
cd svelte-preprocess-autoimport
yarn
yarn dev
```

## Run the example

```bash
yarn link # while still in project's root
cd example
yarn
yarn link svelte-preprocess-autoimport
yarn dev
```

## Run tests

```bash
yarn test
```

Watch mode:

```bash
yarn test -w
```
